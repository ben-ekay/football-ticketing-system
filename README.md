# Football Ticketing System

Football ticket booking system developed using PHP and MySQL.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP / MAMP

## Project Description
The system allows users to register, log in, and book football tickets through a simple web interface. It uses a backend database to manage users and ticket information.

## Features
- User registration and login system
- Ticket booking functionality
- Database integration with MySQL
- Basic session management
- Responsive web design

## Purpose
This project was developed to practice full-stack web development, including backend logic, database management, and user authentication systems.
